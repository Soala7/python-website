Functions

1/5

Functions allow you to organize your code into reusable blocks. Python supports several advanced function features:

  • Default arguments: Parameters with predefined values
  • Keyword arguments: Passing arguments by parameter name
  • Variable-length arguments: *args and **kwargs for flexible parameters
  • Lambda functions: Small anonymous functions
  • Decorators: Modify function behavior without changing code

Example: Function with default and keyword arguments

def greet(name, greeting="Hello", punctuation="!"):
    """Return a personalized greeting"""
    return f"{greeting}, {name}{punctuation}"

# Using default arguments
print(greet("Soala"))  # Hello, Soala!

# Using keyword arguments
print(greet("Alice", punctuation="?"))  # Hello, Alice?

Object-Oriented Programming

2/5

OOP allows you to create classes and objects to model real-world data and behavior. Key concepts include:

  • Encapsulation: Bundling data and methods that work on that data
  • Inheritance: Creating new classes from existing ones
  • Polymorphism: Using a unified interface for different data types
  • Abstraction: Hiding complex implementation details
  • Magic methods: Special methods like __init__, __str__

Example: Basic class with inheritance

class Vehicle:
    def __init__(self, brand, model):
        self.brand = brand
        self.model = model
    
    def info(self):
        return f"{self.brand} {self.model}"

class Car(Vehicle):
    def __init__(self, brand, model, doors):
        super().__init__(brand, model)
        self.doors = doors
    
    def info(self):
        return f"{super().info()} with {self.doors} doors"

my_car = Car("Toyota", "Corolla", 4)
print(my_car.info())  # Toyota Corolla with 4 doors

Modules and Packages

3/5

Python's module system helps organize code into logical units. Key aspects include:

  • Modules: Single Python files containing reusable code
  • Packages: Collections of modules in directories
  • Standard Library: Built-in modules for common tasks
  • pip: Package installer for Python
  • Virtual environments: Isolated Python environments

Example: Creating and using a module

# math_module.py
"""A simple math module with basic operations"""

def add(x, y):
    """Return the sum of two numbers"""
    return x + y

def multiply(x, y):
    """Return the product of two numbers"""
    return x * y

if __name__ == "__main__":
    # Test code when run directly
    print(add(2, 3))  # 5
# main.py
from math_module import add, multiply

result = add(5, 7)
print(result)  # 12

product = multiply(3, 4)
print(product)  # 12

File Handling

4/5

Python provides powerful tools for working with files and directories:

  • File modes: 'r' (read), 'w' (write), 'a' (append), 'b' (binary)
  • Context managers: Using with for automatic resource management
  • Common operations: Reading, writing, appending, seeking
  • File system operations: Using the os and pathlib modules
  • CSV and JSON: Working with structured data formats

Example: Reading and writing files

# Writing to a file
with open("example.txt", "w") as file:
    file.write("Hello, Python World!\n")
    file.write("This is a second line.\n")

# Reading from a file
with open("example.txt", "r") as file:
    content = file.read()
    print(content)

# Appending to a file
with open("example.txt", "a") as file:
    file.write("This line was appended.\n")

Exception Handling

5/5

Robust programs handle errors gracefully using exception handling:

  • Try-except blocks: Catch and handle exceptions
  • Exception hierarchy: Built-in exception classes
  • Custom exceptions: Creating your own exception types
  • Else and finally: Additional control flow
  • Common exceptions: ValueError, TypeError, IOError, etc.

Example: Comprehensive exception handling

def divide_numbers():
    try:
        numerator = float(input("Enter numerator: "))
        denominator = float(input("Enter denominator: "))
        result = numerator / denominator
    except ValueError:
        print("Error: Please enter valid numbers")
    except ZeroDivisionError:
        print("Error: Cannot divide by zero")
    else:
        print(f"Result: {result}")
    finally:
        print("Operation attempted")

divide_numbers()
